## Quick Start Guide
Install and have your USB Rubber Ducky working in less than 5 minutes.
1. Copy the adafruit-circuitpython-raspberry_pi_pico_w-en_US-9.2.1.uf2 file to the root of the Pico (RPI-RP2). The device will reboot and after a second or so, it will reconnect as CIRCUITPY.
